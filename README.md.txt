파이썬 강의를 수강하며 개인적으로 만들어 본 프로그램 모음입니다.
1. 뽀모도로(Pomodoro) 타이머 GUI 프로그램
  - Tkinter 라이브러리를 사용하여 타이머, 버튼, 체크 등 사용자 인터페이스(UI)를 구성하고 화면을 배치
  - window.after() 를 활용하여 1초마다 반복 실행되게 하였고 타이머의 상태(집중/휴식)를 관리
  - 전역 변수를 사용하여 반복 횟수를 추적하고, 짧은 휴식과 긴 휴식을 자동으로 전환

2. 빌보드 차트 기반 Spotify 플레이리스트 생성기
  - BeautifulSoup를 사용하여 특정 날짜의 빌보드 Hot 100 차트 웹 페이지에서 곡 제목들을 정확히 스크래핑 함
  - Spotipy 라이브러리와 Spotify Web API를 연동, OAuth 2.0를 통해 사용자 계정에 접근하게 함
  - 스크래핑한 곡 목록을 Spotify API로 검색하고, 이를 기반으로 사용자 계정에 새로운 플레이리스트를 자동으로 생성 및 곡 추가 

3. 자연어 처리 API를 활용한 운동 기록 자동화
  - Nutritionix API를 활용하여 "30분 달리기"와 같이 입력하면 운동 종류, 시간, 소모 칼로리 등의 정형 데이터로 변환
  - Sheety API를 통해 Google Sheets와 연동, 처리된 운동 데이터를 지정된 시트에 새로운 행으로 POST 요청하여 자동으로 기록
